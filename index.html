• Modo Sugestivo
    O modo sugestivo é o mais simples de todos. Neste modo é apresentado imagens juntamente com notificações de todas as categorias, notificações relacionadas com o lazer, o trabalho, saúde e prática de desporto. Entrar
  • Modo Normal
    No Modo Normal, o utilizador já consegue filtrar as notificações por categorias, ou seja, neste modo o utilizador escolhe uma categoria e a aplicação apenas dá notificações relacionadas com essa categoria. Entrar
  • Modo Intrusivo
    No modo intrusivo existe a possibilidade de criar novas actividades condicionadas a perguntas básicas para definir que tipo de actividade. Após fazer essa escolha serão gerados temporizadores a contabilizar o tempo de cada acção definida no início de cada questão sobre a actividade, sendo despoletado uma notificação caso seja necessário o utilizador parar essa acção mesmo que seja temporáriamente. Para passar as imagens faça swipe. Swipe para a esquerda avança para a imagem seguinte, swipe para a direita volta para a imagem anterior.

Modo normal

Neste modo poderá escolher uma actividade pré definida e serão apresentadas sugestões de acordo com essa actividade.

O modo intrusivo é o mais completo. Permite definir actividades novas e configura-las de acordo com um conjunto de perguntas pré definidas.
